# Changelog

Registro de los módulos entregados durante el desarrollo de Asset Tracking.

## [1.0.0]

### Añadido

- Autenticación JWT y cifrado de contraseñas (PBKDF2).
- Gestión de usuarios, roles y permisos.
- Gestión de personas (estudiantes, docentes, administrativos).
- Gestión de activos tecnológicos con categorías, marcas y modelos.
- Registro de ingresos/salidas y validación de acceso en portería.
- Historial y trazabilidad de movimientos, y panel de alertas.
- Dashboard con estadísticas generales.
- Reportes exportables en PDF y Excel.
- Módulo de auditoría de operaciones.
- Suite de pruebas unitarias (xUnit) e integración continua con GitHub Actions.
